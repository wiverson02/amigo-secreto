# Amigo Secreto

App simples para sortear amigo secreto (HTML, CSS e JavaScript).

```

Uso:
- Digite um nome e clique em "Adicionar".
- Clique em "Sortear" quando tiver pelo menos 2 nomes.
- Clique em "Reiniciar" para limpar tudo.

Observações:
- Projeto somente front-end (sem Node).


Fim.
- Feedback visual ao adicionar um amigo (efeito "flash" na lista de amigos).
